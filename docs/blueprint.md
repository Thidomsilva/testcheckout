# **App Name**: FlexiPay

## Core Features:

- Adjustable Payment Amount: Allow users to manually input the desired payment amount before proceeding to checkout.
- Pix Payment Gateway: Integrate a Pix payment gateway for instant payment processing via QR code or copy-paste.
- Credit Card Payment Gateway: Integrate a credit card payment gateway to accept payments from major credit card providers.
- Payment Confirmation: Display a clear payment confirmation screen after successful transaction, showing amount paid, payment method, and transaction ID.
- Transaction History: Provide users with access to a transaction history, displaying all previous payments with details such as date, amount, and payment method.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to convey trust and security.
- Background color: Light gray (#F0F2F5) for a clean and modern look.
- Accent color: Teal (#008080) for calls to action and important information.
- Body and headline font: 'Inter', a sans-serif font known for its readability and modern design.
- Use clear and recognizable icons for payment methods and transaction status.
- Maintain a simple, uncluttered layout with clear calls to action. Prioritize ease of use on mobile devices.
- Subtle animations for payment confirmations and loading states.
